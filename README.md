# MindStorm

Интерактивная доска для брейншторма в стиле [Obsidian Canvas](https://obsidian.md/canvas). Формат данных — [JSON Canvas](https://jsoncanvas.org/).

**Онлайн:** https://m11nic89co.github.io/mindstorm/

**Репозиторий:** https://github.com/m11nic89co/mindstorm

> Для разработчиков и AI: обязательный контекст — **[CONTEXT.md](./CONTEXT.md)**

---

## Возможности

- Текстовые **карточки** (заголовок + текст), **простой текст** на холсте (цвет и размер) и **группы** с изменением размера
- **Двойной клик** по холсту — новая карточка **серого** цвета (12)
- **Resize группы** — меняется рамка; **выделенные** карточки внутри масштабируются; невыделенные — без изменений
- **Замок на метке группы** — закреплённая группа становится фоном (не двигается и не редактируется); активен только замок; размер иконки = размеру текста метки
- **Размер шрифта** заголовка и текста карточки; **размер метки** группы (до 200 px) — панель справа
- **Ctrl+C / Ctrl+V** — копия карточки или группы рядом (работает при любой раскладке клавиатуры)
- **12 цветов** и название — панель справа при выборе узла
- **Связи** — 8 точек на узел (2 на сторону), подписи, цвет от исходной карточки, анимация к стрелке
- **Undo / Redo** (Ctrl+Z / Ctrl+Shift+Z — любая раскладка)
- Pan / zoom; **ПКМ + рамка** — выделение нескольких узлов
- **Светлая / тёмная тема** — кнопка ☀/☾ в toolbar (сохраняется в браузере)
- **Печать** — вся схема или выделенный фрагмент; A4 альбом, весь холст, читаемый текст, серые подписи на стрелках, без мини-карты
- **RU / EN / ES / 中** — переключатель языка в toolbar
- **Автосохранение** черновика в браузере
- **↺ Демо** — пример схемы; в toolbar первая текстовая кнопка (перед «Текст»)
- **Сохранить / Загрузить / Сначала** — иконки справа; Save пишет в папку **`.mindstorm` + PNG**; папка запоминается
- Совместимость с `.canvas` (Obsidian)

## Быстрый старт

1. Откройте [сайт](https://m11nic89co.github.io/mindstorm/) или `npm run dev`
2. **Двойной клик** по пустому месту — новая **серая** карточка
3. После Undo/Redo: **↺ Демо** → **T Текст** → **+ Карточка** → **◻ Группа**
4. Справа: **Сначала (лист+) → 📂 Загрузить → 💾 Сохранить → 🖨 Печать → ☀/☾ Тема**
5. **Двойной клик** по **верху** карточки — заголовок; по **низу** — текст (редактируются отдельно)
6. **Точки на карточке** (при наведении) — потяните для связи
7. **Клик по карточке, тексту или группе** — название/текст, размер шрифта и цвет справа
8. **Угол группы** — растянуть рамку; **выделенные** карточки внутри масштабируются (Shift+клик)
9. **Замок** на метке группы (справа) — закрепить как фон или снова разрешить редактирование
10. **Ctrl+C / Ctrl+V** — копия выделенной карточки, текста или группы
11. **Сначала** — новая пустая схема (с подтверждением; Undo вернёт)
12. **Сохранить** — PNG по умолчанию (превью); для повторного редактирования выберите `.mindstorm`
13. **Загрузить** — только `.mindstorm` / `.canvas` (PNG открыть как схему нельзя)
14. **🖨** — печать: вся схема или выделенное; A4 альбом, весь холст, читаемый текст
15. **☀ / ☾** — светлая или тёмная тема
16. **↺ Демо** — первая среди текстовых кнопок (перед «Текст»)
17. **RU | EN | ES | 中** — язык интерфейса
18. **Delete** — удалить выделенное
19. **Колёсико** — zoom (сильное отдаление для больших схем)

## Поделиться и установить

**Ссылка для всех:** https://m11nic89co.github.io/mindstorm/

- **Windows / Android:** Chrome или Edge → меню → «Установить MindStorm» (PWA, бесплатно).
- **Донат автору:** кнопка **☕ Донат** внизу экрана (настраивается в `src/config/donate.ts`).

## Формат `.mindstorm` (кратко)

```json
{
  "format": "mindstorm-board",
  "version": 1,
  "title": "моя-схема",
  "savedAt": "2026-06-21T12:00:00.000Z",
  "canvas": { "nodes": [], "edges": [] }
}
```

Старые `.mindshtorm` и Obsidian `.canvas` тоже открываются.
